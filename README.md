This is a Maven project where i have followed a simple folder structure and the scenario is written under a class named as FlipkartTest.java inside a package "com.automationtask" under src/test/java folder.
The complete test script is written in a single class and it can be executed from that specific class.
The screenshots are stored inside a folder "Screenshots" at the project level as per the scenario and the path for the same is passed to the screenshot code.
The library used is selenium-java (v 4.38.0) which is passed as dependency in pom.xml file of the project.
